## 🧠 About the Project

Vibe Coding is an AI-powered node-based automation system.

Users can create and connect nodes such as Name, Age, Address, etc., and build automation flows visually.

In addition, the system integrates an AI Agent powered by :contentReference[oaicite:0]{index=0} :contentReference[oaicite:1]{index=1}, which allows users to simply describe the automation they want, and the AI automatically generates the full workflow for them.

## 🚀 Features

- Node-based visual automation system
- Create and connect custom nodes (Name, Age, Address, etc.)
- Data flow between connected nodes
- Execution engine for automation workflows
- AI Agent powered by :contentReference[oaicite:2]{index=2}
- Generate automations using natural language prompts
- Console output for final processed results

---

## 🤖 AI Agent (Gemini Integration)

The system includes an AI Agent powered by :contentReference[oaicite:3]{index=3}.

Instead of manually creating nodes, users can simply describe what they want, for example:

> "Create an automation that takes name, age, and prints a welcome message"

The AI will:
- Interpret the request
- Generate the required nodes
- Connect them automatically
- Build a ready-to-run automation workflow

---

## 🔄 How It Works

1. User describes automation in natural language OR builds it manually using nodes
2. AI Agent (:contentReference[oaicite:4]{index=4}) can generate the workflow automatically
3. Nodes are created and connected dynamically
4. Data flows through the system
5. Automation is executed
6. Final output is printed in the console

## ⚙️ Installation

1. **Clone the repository**
   ```bash

   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Run the app**
   ```bash
   flutter run
   ```
